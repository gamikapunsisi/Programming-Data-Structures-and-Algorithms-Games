# TSP Game Frontend (React + Vite)

## 1. Clone the Repository

``` bash
git clone https://github.com/Hirushan-N/tsp-frontend
cd tsp-frontend/frontend
```

## 2. Install Node Dependencies

``` bash
npm install
```

## 3. Start the Development Server

``` bash
npm run dev
```

## 4. Open in Browser

``` text
http://localhost:5173
```

## IMPORTANT

The backend must be running at the same time on:

``` text
http://localhost:5000
```

## 5. Stop the Frontend Server

Press:

``` text
CTRL + C
```
